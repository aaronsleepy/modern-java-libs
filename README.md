# modern-java-libs

## 모던 자바 인 액선(8,9,10) 스터디 정리

## Java 8
### 요구사항
* 간결한 코드
* 멀티코어 프로세서의 쉬운 활용 (병렬처리)


### 신규 기능
* 스트림 API
    * 선언형 프로그래밍 like SQL
    * 병렬연산 지원

* 메서드에 코드를 전달(메서드 참조 & 람다)
    * 동작 파라미터화(behavior parameterization) 
    * 7까지는 익명 클래스를 이용
    * 람다 -> 함수형 프로그래밍 가능

* 인터페이스의 디톨트 메서드


### 함수형 프로그래밍
* 무엇을 할 것인가를 기술하며 어떻게 할 것인가는 API에게 맡긴다


### Misc
* Optional<T>
